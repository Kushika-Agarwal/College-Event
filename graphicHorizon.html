Graphics Horizon

About the Event

Graphics Horizon is a coding challenge focused on Computer Graphics using C language. Participants will be given a problem statement related to graphics programming in C and will have 60 minutes to develop a solution using their coding and design skills. This event aims to test logical thinking, creativity, and efficiency in implementing graphical concepts using C. <

Team Size

  • Individual participation

Guidelines

  • Eligibility: Open to all students of BCA/B.Tech/MCA.
  • Team Compostition:Each participant competes individually.
  • Duration:60 minutes

Rules:

  • Task Overview: Participants will receive a problem statement related to graphics designing in C language.
  • Submission Format: The solution must be coded in C language, and the output should be displayed in a graphical format.
  • Presentation: Participants must complete their code within the given 60-minute time limit.
  • Software: Participants can use Turbo C++ or Dev-C++ for implementation.

Prize Distribution-

  • Ist Place- Gold Medal and Certificate
  • IInd Place -Silver Medal and Certificate
  • IIIrd Place - Bronze Medal and Certificate
