About the Event

C Code Chaos is an electrifying challenge where you must decode the madness, rearrange scrambled C code, and prove your programming prowess—before time runs out!One minute. One jumbled code. One chance to fix the chaos.
Do you have what it takes to emerge as the ultimate Code Conqueror?

Team Size

  • 1 Members

Prizes & Certificates:

  • Winners and runners-up will receive certificates & prizes.
  • Participation certificates will be provided to all.

Round Structure:

Round 1: Basic Jumbled Code

  • Participants will be given shuffled or incomplete code snippets and must rearrange them to form a working program.
  • The output must match the expected result.
  • Top performers will proceed to the next round.
    • Round 2: Debugging & Optimization
  • Participants will be provided with faulty or inefficient code and must debug & optimize it.
  • Bonus points for improving efficiency.
    • Final Round: Logic Building with Jumbled Code
  • A complex logic-based program will be provided in jumbled form.
  • Teams must arrange the code logically and ensure proper execution.
  • The fastest correct submission wins.

Scoring Criteria:

  • Accuracy: Correct output and logical structure.
  • Time Taken: Faster solutions score higher.
  • Code Efficiency: Optimized code earns extra points.
  • Error Handling: Proper error handling is rewarded.

Disqualification Criteria:

  • Any form of plagiarism or cheating will lead to immediate disqualification.
  • Team switching after registration is not allowed.
  • Violation of event timing (late submissions, failure to report to the event) may lead to disqualification.

Guidelines:

  • Participation: Individual participation only. No teams allowed.
  • Eligibility: Open to all students across all branches and years.
  • Time Limit: Each participant will get 1 minute to solve each question.
  • Question Format: Participants will be given jumbled code snippets that need to be rearranged into the correct logical order.
  • Evaluation: Answers will be judged based on correctness and completion within the given time.
  • No External Help: Use of internet, mobile phones, or external coding resources is strictly prohibited during the competition.
  • Round Structure: The event may have multiple rounds, with increasing difficulty in each round.
  • Tiebreaker: In case of a tie, an additional rapid-fire coding challenge will be conducted.
  • Disqualification: Any form of misconduct, cheating, or rule violation will result in immediate disqualification.
  • Decision Authority: The event coordinators’ decision will be final and binding in all matters.
  • Platform: Wordwall.net
