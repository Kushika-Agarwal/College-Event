Web Rachaiyta

About the Event

Participants have to develop a web application on a given topic on the same day, which will be evaluated on the following features:

  • GUI
  • Look and feel
  • Content Management
  • CSS and Client-Side & Server-Side Scripting

Team Size

  • 2-4

Guidelines

  • Eligibility - Any Stream
  • Team Composition - 2-4
  • Duration - 2 hours

Rules

  • The competition requires a minimum 5-page website developed on the spot.
  • Participants may use any web development software (e.g., Dreamweaver, Visual Studio) or languages (HTML, JavaScript, etc.).
  • The website must follow a given template for presentation to the judges.
  • The first page must be the homepage containing a logo/image relevant to the topic.
  • Participants must bring their own laptops and necessary software/tools.
  • The last page must be an "Information Hub" detailing the web designer's name, tools used, and references.

Prize Distribution

  • 1st Place - Gold Medal
  • 2nd Place - Silver Medal
  • 3rd Place - Bronze Medal
